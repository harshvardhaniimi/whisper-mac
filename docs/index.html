Quick Install

Run this in Terminal:

Then right-click the app in Applications ‚Üí "Open" (first time only)

Manual Installation

  1. Download the zip file and extract it
  2. Move WhisperMac.app to your Applications folder
  3. Important: Open Terminal and run:
    xattr -cr /Applications/WhisperMac.app
  4. Right-click WhisperMac.app and select "Open"

Setup Permissions

  1. Grant Accessibility permissions:
    System Settings ‚Üí Privacy & Security ‚Üí Accessibility ‚Üí Enable WhisperMac
  2. Grant Microphone access when prompted
‚ö†Ô∏è Why Terminal commands? macOS quarantines apps downloaded from the internet. Since WhisperMac isn't notarized with Apple ($99/year), you need to remove the quarantine flag. This is safe for open-source apps you trust.

Usage

  1. Click the waveform icon in your menu bar
  2. Or press Ctrl twice quickly from anywhere
  3. Speak your text
  4. Press Ctrl twice again to stop
  5. Text appears at your cursor!
